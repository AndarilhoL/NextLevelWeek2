# Executar o Projetinho s2


```bash
# Clonar o repositório
git clone https://github.com/AndarilhoL/NextLevelWeek2

# Entrar no diretório
cd NLW_2

# Baixar as dependências
npm i

# Executar o servidor
npm run dev
```

Após o passo a passo anterior, abra o seu navegador e acesse `http://localhost:5500/`
